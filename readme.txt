Adds the F1 Minute audio player to a widget-enabled WordPress theme.

== Description ==

The F1 Minute audio player is a simple widget that plays the most recent episodes of the sixty second Formula One podcast 'F1 Minute'.  New shows are released each and every weekday, which means you get lots of free and fresh audio content on your web site.

NB: Options are available to alter the shape and size of the player.  The player requires that listeners have Flash Player 8 or above installed.

== Installation ==

1. Upload the 'f1minute-player' folder to your '/wp-content/plugins/' directory.
2. Activate the player through the 'Plugins' menu in WordPress.
3. View your site and ensure player is working as expected.

Options can be configured from the 'Presentation > Widgets' menu within WordPress.

== Frequently Asked Questions == 

= Question 1 =
Aside from width and height values, can I change the look of the player?

Answer 1
Yes, the player can be further styled using CSS properties, like so:

.widget_f1minute_player {
	background-color: #fff;
	color: #ccc;
	/* additional CSS properties go here */
}
